# Development Roadmap

## Phase 1: Core MVP

- Authentication and profile management.
- Cycle entry and phase prediction.
- Dashboard cards and basic history pages.
- Symptom and mood tracking.

## Phase 2: Wellness Modules

- Nutrition and exercise recommendation engines.
- Water and sleep trackers.
- Daily journal.
- Calendar module.

## Phase 3: Analytics

- Cycle length trend charts.
- Mood distribution charts.
- Symptom frequency chart.
- Period regularity score.
- Health alerts and safety disclaimer.

## Phase 4: Reporting

- Complete history view.
- PDF export.
- More structured report sections for clinicians or personal review.

## Future Enhancements

- Email delivery for reset links.
- Flyway or Liquibase migrations.
- Doctor appointment reminders and medication reminders.
- CSV export.
- Testcontainers integration tests.
- Role-based admin dashboard for managing recommendation content.
- Push notifications for predicted period and hydration reminders.
- More advanced analytics using rolling averages and cycle variance.
- Optional wearable import for sleep and activity data.
- Localization and accessibility audit.
- Privacy controls for data deletion and export.
